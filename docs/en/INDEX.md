### By Use Case

**Building a new feature:**
1. Start with [Multi-Step Planning](advanced-skills/multi-step-planning.md)
2. Use [Code Understanding](core-skills/code-understanding.md) to understand context
3. Apply [File Operations](core-skills/file-operations.md) to implement
4. Validate with [Terminal Operations](core-skills/terminal-operations.md)

**Debugging an issue:**
1. Use [Search & Navigation](core-skills/search-navigation.md) to locate problem
2. Apply [Code Understanding](core-skills/code-understanding.md) to analyze
3. Fix with [File Operations](core-skills/file-operations.md)
4. Test with [Terminal Operations](core-skills/terminal-operations.md)

**Refactoring code:**
1. Analyze with [Code Understanding](core-skills/code-understanding.md)
2. Plan with [Multi-Step Planning](advanced-skills/multi-step-planning.md)
3. Execute with [File Operations](core-skills/file-operations.md)
4. Verify changes work correctly
